HTML

HTML (Hypertext Markup Language) is the standard markup language used for creating and structuring web pages. It provides a set of tags and elements that define the structure and content of a webpage. HTML documents consist of a series of nested elements, forming a hierarchical structure.

Bootstrap

Bootstrap is a popular open-source front-end framework that provides a collection of pre-built HTML, CSS, and JavaScript components. It is designed to simplify and speed up the process of creating responsive and mobile-first websites and web applications.

Node.js Server

Node.js is an open-source, cross-platform JavaScript runtime environment built on Chrome's V8 JavaScript engine. It allows developers to run JavaScript code outside of a web browser, enabling server-side execution of JavaScript. Node.js provides a scalable, event-driven architecture that makes it suitable for building fast and efficient network applications.
